### Why?

Pepper allows you to manage your shortlinks with ease. It is built with Next.js and Vercel serverless functions.

### Tech Stack
 - Next.js
 - Edge Functions
 - Tailwind CSS

### Installation

1. Clone the repo
```bash
git clone
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```
